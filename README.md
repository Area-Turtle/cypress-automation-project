# cypress-automation-project
is a cypress automation for a example template website
Demo Website is from OWASP Juice-Shop 

To install clone OWASP Juice-Shop Repo (git clone https://github.com/juice-shop/juice-shop)
check if version of node/nvm is compatable (update node via nvm to at minimum to v20.x, v22.x and v24.x > or latest LTS version)
run the command npm install (first time only)
run command npm start to launch the website on localhost > http://localhost:3000
